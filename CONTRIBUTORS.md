## How to Contribute

We welcome contributions! Please:

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/AmazingFeature`)
3. Commit your changes (`git commit -m 'Add AmazingFeature'`)
4. Push to the branch (`git push origin feature/AmazingFeature`)
5. Open a Pull Request

## Attribution

If your contribution doesn't automatically appear in GitHub's contributors list, please ensure that:
- Your commit email matches an email address verified on your GitHub profile
- You configure your local Git with: `git config user.email "your-verified-email@users.noreply.github.com"`

Thank you for contributing! 🚀
